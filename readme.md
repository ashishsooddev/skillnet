# Skillnet \- Professional Skill Networking Platform



## 📋 Overview

Skillnet is a lightweight front-end skill communication platform. It focuses on personal skill display and user interaction, featuring a clean and professional web interface. The platform supports user login and enables developers to communicate and exchange ideas with peers in the industry.\.

## ⚙️ Core Features

- ✅ **Login program** \- Users can log in after entering the correct email address and password

- ✅ **Email/password error reminder** \- Users will receive a notification if they enter the wrong email or password

- ✅ **User account homepage** \- Users can receive random referrals
